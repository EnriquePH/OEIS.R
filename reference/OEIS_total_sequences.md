# Retrieve the total number of sequences in the OEIS database

Retrieve the total number of sequences in the OEIS database

## Usage

``` r
OEIS_total_sequences()
```

## Value

An integer with the total number of sequences, or `NA_integer_` if the
extraction fails or there is no internet connection.

## See also

- [`OEIS_sequence()`](https://enriqueph.github.io/OEIS.R/reference/OEIS_sequence.md)

## Examples

``` r
if (FALSE) { # \dontrun{
OEIS_total_sequences()
} # }
```
